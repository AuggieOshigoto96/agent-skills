// The toolkit

Five skills. Every security workflow.

01 / SOC Operations

Alert Triage

Paste raw alert JSON from Defender, Splunk, or Sentinel. Get back a structured verdict with MITRE ATT&CK mapping, reassessed severity, and a SOAR-ready action payload in seconds.

MITRE ATT&CK Splunk Defender XSOAR
Stack: Python · JSON normalization · LLM reasoning
alert-triage — claude
02 / Detection Engineering

Sigma Rule Converter

Port community Sigma rules into your SIEM of choice. Handles Splunk SPL, Sentinel KQL, MDE KQL, Elastic, and CrowdStrike — with field mapping notes and tuning suggestions preserved.

Sigma SPL KQL Elastic
Stack: Python · PyYAML · Cross-SIEM field mapping
sigma-converter — claude
03 / AI Security

Prompt Injection Tester

A red-team framework that generates OWASP LLM Top 10 payloads, runs them against your endpoint, and grades responses with heuristics plus LLM-as-judge. Produces audit-ready reports mapped to LLM01, LLM02, LLM06, and LLM07.

OWASP LLM Top 10 Red Team MCP Security ISO 42001
Stack: Python · Anthropic API · Heuristics + LLM-as-judge
prompt-injection-tester — claude
04 / Cloud Security

AWS IAM Reviewer

Reviews AWS IAM policies for least-privilege violations and ranks findings by blast radius. Catches classic privilege escalation paths like iam:PassRole chains, policy version overwrites, and permissive trust policies.

AWS IAM Least Privilege Privilege Escalation Cloud Audit
Stack: Python · AWS policy AST · Blast-radius scoring
aws-iam-reviewer — claude
05 / Cloud Security

GuardDuty Finding Triage

Cloud-native sibling to Alert Triage. Takes raw AWS GuardDuty findings and produces an incident verdict with MITRE ATT&CK for Cloud mapping, context-aware severity reassessment, and runnable remediation commands. Handles 50+ finding families across EC2, IAM, S3, and EKS.

AWS GuardDuty MITRE ATT&CK Cloud SOAR Cloud IR
Stack: Python · boto3 · Security Hub ASFF · AWS CLI
guardduty-triage — claude
// The approach

Scripts for determinism. Claude for judgment.

Each skill follows the same design: Python handles the deterministic work — parsing, validation, API calls, linting. Claude handles what LLMs do well — MITRE mapping, field translation, blast-radius reasoning, verdict synthesis. The result is reproducible, structured, production-ready output. No vibes. No guessing. No generic AI slop.
